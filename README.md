# Monica Energy v3 Offline Project

## Main updates
- Login now opens a clean dashboard first.
- Added separate pages for Customer Registration, Customer Search, Repayment, Admin Audit, Fuel Prices, App Settings, and Credit History.
- Brand appears at the top of every page.
- Current balance and credit limit are shown in customer profile.
- Daily / monthly / yearly audit is included.
- Credit history has a detailed next page.
- Dates are displayed as DD/MM/YYYY.
- Import / export backup is supported in App Settings.

## Offline
All data is kept in localStorage so the app works without internet.
